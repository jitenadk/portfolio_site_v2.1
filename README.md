## Getting Started

1. Clone the repo:

   ```bash
   git clone https://github.com/jitenadk/portfolio_site_v2.1.git
   cd portfolio_site_v2.1
   ```

2. Install dependencies:

   ```bash
   npm install
   # or yarn install
   ```

3. Start development server:

   ```bash
   npm run dev
   # or yarn dev
   ```

4. Visit [http://localhost:3000](http://localhost:3000).
